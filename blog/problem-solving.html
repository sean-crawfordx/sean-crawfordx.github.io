🧠 From Frustration to Small Wins: How I’m Learning to Code


🧱 A Time I Was Blocked on a Simple Problem

I was trying to link my JavaScript file to my HTML file. It seemed like it should be simple. I’d seen it done in tutorials and assumed it would work right away.
But when I opened the browser, nothing happened. No errors. Just nothing. I started guessing what was wrong. I changed file names, rearranged folders, and refreshed my browser about a hundred times. Eventually, I asked AI for help (thank you ChatGPT!) and realized I had made a small typo in the path to my JavaScript file.
Such a tiny mistake, but it broke everything. I felt frustrated, annoyed, and a bit embarrassed. But I also learned that in code, little things really matter. Now, I double-check my file paths and start with the basics when things don’t work.


🎯 A Time I Solved Something Elegantly

I wanted to make a spinning cube using Three.js. I saw an example online and thought it looked cool. The code was already written in one block, but I wanted to try splitting it into proper files, like a real project setup. At first, I didn’t understand what everything was doing, but I used AI to help explain the parts and guide me through the process.
I created separate HTML and JS files, made sure the module imports were correct, and used npx vite to serve the project.
It didn’t feel like I was doing anything super technical, but just getting the structure right and seeing the cube spin smoothly in the browser made me feel proud. I realized that even organizing existing code and getting it running is a step forward in learning.

🙋‍♂️ When I Was Reluctant to Ask for Help

The very first time I started panicking about time-boxes. I felt too embarrassed to ask for help on Discord, where everyone could see.
So instead, I messaged Joseph directly for some of his wisdom — he’s a super-communicator and just a really intelligent, supportive person.
Looking back, I realise there’s no shame in asking publicly. We’re all learning, and someone else probably has the same question. But in that moment, I just wanted help without feeling exposed.
I’ve been in a trade for nearly 30 years. I’ve always been the kind of person who will use any tool I can get my hands on to make my job easier.
And now that I’m learning tech, I take the same approach. I’ll ask for help, use AI, dig through forums — whatever it takes.

My goal is to make the computer work for me, not the other way around. 🛠️💻
